# Web Mapping

## 2026

- [Standalone Web Maps Workshop - FOSS4G 2025](https://mizmay.github.io/standalone_web_maps_foss4g2025/workshop/welcome/): Workshop teaching creation of standalone web maps using Overpass and MapLibre, then deploying to GitHub Pages without a server.
  - Keywords: web mapping, MapLibre, GitHub Pages, OSM, FOSS4G, JavaScript

## 2025

- [Mapweaver](https://dosull.github.io/mapweaver/app/): A web app for making tiled and woven maps for multivariate and bivariate data visualization.
  - Keywords: web mapping, cartography, bivariate, tiled, visualization, JavaScript

## 2024

- [QGIS Web Viewer Plugin](https://plugins.qgis.org/plugins/QuickWebViewer/): QGIS plugin for publishing maps as web viewers.
  - Keywords: web mapping, QGIS, plugin, viewer, JavaScript

- [Cloud Native Geospatial Map](https://www.postholer.com/articles/Next-Generation-Interactive-Maps): Article on next-generation interactive maps using cloud-native geospatial technologies.
