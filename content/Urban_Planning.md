# Urban Planning

## 2025

- [How Bangalore Uses the Metro](https://diagramchasing.fun/2025/how-bangalore-uses-the-metro): Data analysis of Bangalore metro ridership patterns using RTI data showing the Yellow Line's 16% ridership increase and Lalbagh station's 40% surge after August 2025 opening.
  - Keywords: urban planning, Bangalore, metro, transit, ridership, data analysis

## 2024

- [NetaScore - Walkability using OpenStreetMap](https://github.com/plus-mobilitylab/netascore/tree/v1.1.1): Open-source toolbox assessing bikeability and walkability of road networks using OpenStreetMap data for sustainable mobility planning.
  - Keywords: urban planning, walkability, bikeability, OpenStreetMap, Python, mobility

- [Spatial Configuration of Green Space and Air Temperature](https://www.sciencedirect.com/science/article/pii/S0169204624001208): Paper on how urban green space shape and spatial configuration affect air temperature and urban cooling.
  - Keywords: urban planning, green space, urban heat island, air temperature, land cover

## Earlier

- [Zone Builder FOSS4G 2021](https://github.com/zonebuilders/zonebuilder): Tool for building urban zones for transport and planning analysis.
  - Keywords: urban planning, zoning, transport, GIS, R, Python

- [Pedestrian Environment Index (PEI)](https://www.sciencedirect.com/science/article/abs/pii/S0966692314001343): Paper on developing and applying the Pedestrian Environment Index for walkability assessment.
  - Keywords: urban planning, walkability, pedestrian, index, GIS

- [Walkability and the 15-Minute City Model - Milan](https://transformtransport.org/research/livable-streets/walkability-and-the-15-minute-city-model-an-integrated-approach-for-the-city-of-milan/): Integrated walkability and 15-minute city analysis for Milan.
  - Keywords: urban planning, walkability, 15-minute city, Milan, GIS

- [Sidewalk Widths NYC](https://github.com/meliharvey/sidewalkwidths-nyc): Python tool for measuring and mapping sidewalk widths in New York City using GIS data.
  - Keywords: urban planning, sidewalk, width, NYC, Python, GIS

- [NRCan Geo Sim Processing](https://github.com/NRCan/geo_sim_processing): Natural Resources Canada tools for geometric simplification and generalization processing.
  - Keywords: urban planning, geometry, simplification, GIS, NRCan, Python

- [neatnet - Street Network Simplification](https://github.com/uscuni/neatnet): Python toolkit that simplifies street network geometry by removing dual carriageways and roundabouts, replacing them with morphologically accurate centerline representations.
  - Keywords: urban planning, street network, OpenStreetMap, simplification, dual carriageway, Python

- [Madina - Urban Network Analysis](https://github.com/City-Form-Lab/madina): Python library streamlining urban network analysis and modeling of pedestrian and bicycle trips, with tools for network topology, accessibility analysis, and pedestrian flow simulation.
  - Keywords: urban planning, network analysis, pedestrian, bicycle, accessibility, Python
