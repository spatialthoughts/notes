# Cartography

## 2025

- [Weaving Space: Multivariate Maps](https://dosull.github.io/weaving-space/presentations/FOSS4G-Nov-2025/#/title-slide): FOSS4G 2025 presentation introducing `weavingspace`, a Python module using tiling and weaving techniques to combine multiple choropleth maps into single multivariate visualizations.
  - Keywords: cartography, multivariate, choropleth, Python, visualization, FOSS4G

- [Lonboard Visualization - Climate Projection](https://github.com/samarchie/foss4g-auckland-2025-lonboard/): FOSS4G Auckland 2025 notebooks demonstrating high-performance geospatial visualization with Lonboard using GPU-accelerated WebGL rendering.
  - Keywords: cartography, visualization, Lonboard, WebGL, GPU, FOSS4G, Python

## 2024

- [Map Warper - Online Georeferencer](https://mapwarper.net/): Free, open-source online tool for georeferencing historical maps and overlaying them on modern maps.
  - Keywords: cartography, georeferencing, historical maps, online tool

- [Create City Map Print Poster from OSM](https://github.com/originalankur/maptoposter): Python tool generating beautiful minimalist map posters for any city from OpenStreetMap data with customizable themes and high-resolution export.
  - Keywords: cartography, OSM, poster, city maps, Python, visualization

## Earlier

- [Mapschool - Choropleth Maps](https://mapschool.io/#choropleth): Mapschool guide on choropleth and thematic map design theory.
  - Keywords: cartography, choropleth, thematic maps, education

- [Joshua Stevens Bivariate Choropleth Maps](https://www.joshuastevens.net/cartography/make-a-bivariate-choropleth-map/): Tutorial by Joshua Stevens on creating bivariate choropleth maps.
  - Keywords: cartography, bivariate, choropleth, design

- [Learn Cartography Guide by Alex McPhee](https://quercusmaps.com/pages/you-should-make-maps): Guide on learning cartography and why you should make maps.
  - Keywords: cartography, learning, education, maps

- [Color Ramp for Elevation Maps](https://cartographicperspectives.org/index.php/journal/article/download/cp69-patterson-jenny/pdf): Paper on designing effective color ramps for elevation visualization.
  - Keywords: cartography, color, elevation, DEM, design
