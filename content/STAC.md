# STAC (SpatioTemporal Asset Catalog)

## 2025

- [S2 Median Composite of Ireland with Coiled](https://github.com/ahwallace/semantic-eo/blob/main/src/semanticeo/s2_processing.py): Python script for creating Sentinel-2 median composites using Coiled (Dask cloud) and STAC. *STAC, Sentinel-2, composite, Dask, Coiled, cloud-native, Python*

- [odc-stac vs stackstac comparison](https://discourse.pangeo.io/t/comparing-odc-stac-load-and-stackstac-for-raster-composite-workflow/4097/33): Pangeo forum discussion comparing odc-stac and stackstac for raster composite workflows. *STAC, odc-stac, stackstac, Pangeo, Python, comparison*

## 2024

- [STOP and STAC series](https://www.geocorner.net/post/are-you-still-downloading-satellite-images-stop-and-stac-part-2): Series of blog posts explaining why to use STAC instead of downloading satellite images, with practical examples. *STAC, cloud-native, satellite data, remote sensing*

- [Microsoft Planetary Computer STAC API Example](https://medium.com/@geonextgis/getting-started-with-microsoft-planetary-computer-stac-api-67cbebe96e5e): Tutorial on using the Microsoft Planetary Computer STAC API with platform filters for Landsat 8/9 data access. *STAC, Planetary Computer, Landsat, Microsoft, Python*

- [Bitmasking Sentinel-2 with STAC](https://discourse.pangeo.io/t/comparing-odc-stac-load-and-stackstac-for-raster-composite-workflow/4097/14): Pangeo forum discussion on Sentinel-2 bitmask handling in STAC-based raster workflows. *STAC, Sentinel-2, bitmasking, Pangeo, Python*

## Earlier

- [PySTAC QuickStart](https://pystac.readthedocs.io/en/latest/quickstart.html): Official quickstart guide for PySTAC Python library for working with STAC catalogs. *STAC, PySTAC, Python, catalog, satellite data*

- [element84 Earth Search STAC](https://www.element84.com/earth-search/): Element84's open STAC search API for searching and accessing satellite imagery. *STAC, element84, Earth Search, satellite data, API*

- [Access Satellite Data using Python (Carpentries)](https://carpentries-incubator.github.io/geospatial-python/19-access-data/index.html): Carpentries lesson on accessing satellite data via STAC using Python. *STAC, Python, satellite data, education, Carpentries*

- [Sentinel-1 ARD STAC](https://sentinel.esa.int/web/sentinel/sentinel-1-ard-normalised-radar-backscatter-nrb-product): ESA documentation on Sentinel-1 Analysis Ready Data (ARD) NRB product available via STAC. *STAC, Sentinel-1, ARD, SAR, ESA*

- [CBERS AWS STAC](https://twitter.com/liporace/status/1570029005692190720): CBERS (China-Brazil Earth Resources Satellite) data available via AWS STAC catalog. *STAC, CBERS, AWS, Brazil, satellite data*
