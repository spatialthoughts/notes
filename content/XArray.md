# XArray

## 2025

- [EOPF XArray Notebooks - Flood Mapping Valencia](https://eopf-toolkit.github.io/eopf-101/06_eopf_zarr_in_action/64_flood_mapping_valencia.html): EOPF toolkit notebooks demonstrating Otsu thresholding, connected components, and apply_ufunc for flood mapping using ZARR data.
  - Keywords: XArray, ZARR, flood mapping, EOPF, Sentinel, apply_ufunc

- [NDVI Prediction with Dask XArray apply_ufunc](https://github.com/keves1/vegetation-health-monitor/blob/main/pipelines/ml_inference/inference_pipeline.py): Python pipeline using Dask and XArray's apply_ufunc for scalable NDVI-based vegetation health monitoring and ML inference.
  - Keywords: XArray, Dask, NDVI, apply_ufunc, machine learning, vegetation

- [Sentinel-1 ZARR ARD Notebook for Change Detection](https://github.com/bopen/edh-learning/blob/main/notebooks/50-use-case-sentinel-1-small-area.ipynb): XArray-based notebook for Sentinel-1 SAR change detection using ZARR-format Analysis Ready Data.
  - Keywords: XArray, Sentinel-1, ZARR, change detection, SAR, ARD

- [Project Pythia - Spectral Clustering with XArray](https://projectpythia.org/landsat-ml-cookbook/): XArray-based geospatial education resource covering spectral clustering and machine learning on Landsat data.
  - Keywords: XArray, Landsat, spectral clustering, machine learning, education

- [Cloud-native Geospatial Data Cube Workflows](https://e-marshall.github.io/cloud-open-source-geospatial-data-cube-workflows/introduction.html): Jupyter notebooks demonstrating cloud-native geospatial data cube workflows using XArray, XVec, and the Pangeo ecosystem.
  - Keywords: XArray, data cube, cloud-native, Pangeo, Python, XVec

- [Landsat STAC XArray Tutorials - AWS Cloud](https://code.usgs.gov/eros-user-services): USGS EROS tutorials for accessing Landsat data via STAC and processing with XArray on AWS cloud.
  - Keywords: XArray, STAC, Landsat, AWS, USGS, cloud-native

- [Climate Data Earth Data Hub](https://github.com/bopen/edh-learning/blob/main/notebooks/00-getting-started.ipynb): Getting started notebook for the Earth Data Hub (EDH) climate data platform using XArray.
  - Keywords: XArray, climate, ERA5, data hub, Copernicus, Python
