# Google Earth Engine (GEE)

## 2025

- [Human degradation of tropical moist forests](https://www.nature.com/articles/s41586-024-07629-0): Nature paper on quantifying human degradation of tropical forests using GEE with GEDI kernel analysis. GEE Code: https://zenodo.org/records/11235618
  - Keywords: GEE, deforestation, forest degradation, GEDI, tropical, lidar

- [Precipitation Dataset Comparison in GEE](https://www.researchgate.net/publication/380929888_Evaluation_of_Precipitation_Datasets_Available_in_Google_Earth_Engine_on_a_Daily_Basis_for_Czechia): Paper evaluating multiple precipitation datasets available in GEE on a daily basis. [Code](https://github.com/palubad/Precipitation_GEE)
  - Keywords: GEE, precipitation, CHIRPS, rainfall, comparison, evaluation

## 2024

- [Gap Filling and Savitzky-Golay Filtering for Landsat time series](https://github.com/ChenY04/GEE/blob/main/GF-SG): GEE code for reconstructing Landsat time series by fusing MODIS and Landsat using gap-filling and Savitzky-Golay filtering.
  - Keywords: GEE, Landsat, MODIS, time series, gap filling, Savitzky-Golay

- [Atmospheric Correction Sentinel-2 and Landsat in GEE - SIAC](https://github.com/MarcYin/SIAC_GEE): GEE implementation of SIAC atmospheric correction for Sentinel-2 and Landsat imagery.
  - Keywords: GEE, atmospheric correction, Sentinel-2, Landsat, Python
