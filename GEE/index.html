    2025¶

    • Keywords: GEE, deforestation, forest degradation, GEDI, tropical, lidar

    • Precipitation Dataset Comparison in GEE: Paper evaluating multiple precipitation datasets available in GEE on a daily basis. Code

    • Keywords: GEE, precipitation, CHIRPS, rainfall, comparison, evaluation

    • Gap Filling and Savitzky-Golay Filtering for Landsat time series: GEE code for reconstructing Landsat time series by fusing MODIS and Landsat using gap-filling and Savitzky-Golay filtering.
